## 架构概览

GNS3 Copilot 使用模块化架构构建，包含以下关键组件：

```
┌─────────────────────────────────────────────────────────────┐
│                      表现层                                   │
│                   (Streamlit Web UI)                        │
└───────────────────────┬─────────────────────────────────────┘
                        │
┌───────────────────────▼─────────────────────────────────────┐
│                    智能体层 (LangGraph)                       │
│  - 意图识别                                                    │
│  - 工具选择与调用                                              │
│  - 对话管理                                                    │
└───────────────────────┬─────────────────────────────────────┘
                        │
┌───────────────────────▼─────────────────────────────────────┐
│                     工具集成层                                │
│  - GNS3 API 工具                                              │
│  - 配置工具 (Nornir/Netmiko)                                 │
│  - 显示工具                                                   │
└───────────────────────┬─────────────────────────────────────┘
                        │
┌───────────────────────▼─────────────────────────────────────┐
│                    GNS3 客户端层                             │
│  - GNS3 服务器通信                                             │
│  - 项目管理                                                    │
│  - 拓扑操作                                                    │
└───────────────────────┬─────────────────────────────────────┘
                        │
┌───────────────────────▼─────────────────────────────────────┐
│                      GNS3 Server API                         │
└─────────────────────────────────────────────────────────────┘
```

## 关键技术

- **前端 UI**: Streamlit
- **AI 框架**: LangGraph + LangChain
- **网络自动化**: Nornir + Netmiko + Telnetlib3
- **GNS3 集成**: gns3fy（自定义增强版）
- **数据持久化**: SQLite
- **LLM 支持**: OpenAI、DeepSeek、Anthropic、XAI、OpenRouter 等
